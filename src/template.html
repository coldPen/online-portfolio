Sites & Applications Web

Mon but : servir au mieux votre activité en lui associant un site web fiable, optimisé pour vos besoins et aussi simple d'usage que possible.

Passionné par le développement web, tous mes travaux bénéficient d'une veille technique constante. J'accorde un soin particulier à l'accessibilité, la réactivité et la maintenabilité de mes sites et applications.

Sites ultra-réactifs

Je développe votre site avec Gatsby, un outil pensé pour le modèle JAMstack et optimisé pour offrir le meilleur degré de réactivité à vos utilisateurs.

Grâce à l'architecture JAMstack, votre site vitrine ou e-commerce est plus rapide, mieux sécurisé et moins cher qu'un site conçu avec un CMS comme WordPress ou Shopify.

Applications React

Je développe des applications web avec React et Redux, outils plébiscités qui permettent d'intégrer votre logique métier dans une interface web à l'architecture robuste, alimentée via une API REST ou GraphQL.

Design et intégration

Votre site est affiché de façon optimale sur toutes les tailles de terminaux, à l'aide d'outils comme Sass ou Tailwind CSS qui facilitent la mise en page du site et ses modifications.

Le projet

WeCashUp est un service de paiement en ligne offrant l'interopérabilité des services bancaires d'abord sur le continent africain, où une grande partie des utilisateurs ne disposent pas d'un compte bancaire et comptent ainsi sur les services offerts par les services de téléphonie aux fonctionnalités limitées, puis partout ailleurs.
Le formulaire est ainsi exhaustif et offre un nombre conséquent de moyens de paiement dans de nombreux pays. Il intègre également le Split Payment, une fonctionnalité de partage du paiement avec d'autres personnes.

Les outils

jQuery, Material Design Lite, International Telephone Input
